## Входные данные
Вам даны ***n*** чисел. Постройте какое-нибудь BST с этими числами.

## Выходные данные
Выведите BST в формате из предыдущей задачи.

## Пример
входные данные:
```
3
1 2 3
```
выходные данные:
```
3
2 2 3
1 -1 -1
3 -1 -1
1
```
